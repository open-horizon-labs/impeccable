# Impeccable

1 skill, 17 commands, and curated anti-patterns for impeccable frontend design.

> **Packaging fork** of [pbakaus/impeccable](https://github.com/pbakaus/impeccable). All credit to [Paul Bakaus](https://www.paulbakaus.com).

## Install

```bash
npx skills add open-horizon-labs/impeccable
```

## What's Included

**frontend-design** - Comprehensive design skill with references for typography, color, spacing, motion, interaction, responsive design, and UX writing.

**17 commands:**

| Command | What it does |
|---------|--------------|
| `/teach-impeccable` | One-time setup: gather design context, save to config |
| `/audit` | Technical quality checks (a11y, performance, responsive) |
| `/critique` | UX design review: hierarchy, clarity, emotional resonance |
| `/normalize` | Align with design system standards |
| `/polish` | Final pass before shipping |
| `/simplify` | Strip to essence |
| `/clarify` | Improve unclear UX copy |
| `/optimize` | Performance improvements |
| `/harden` | Error handling, i18n, edge cases |
| `/animate` | Add purposeful motion |
| `/colorize` | Introduce strategic color |
| `/bolder` | Amplify boring designs |
| `/quieter` | Tone down overly bold designs |
| `/delight` | Add moments of joy |
| `/extract` | Pull into reusable components |
| `/adapt` | Adapt for different devices |
| `/onboard` | Design onboarding flows |

**Anti-patterns** - Explicit guidance on what to avoid: overused fonts, gray on color, pure black/white, nested cards, bounce easing.
